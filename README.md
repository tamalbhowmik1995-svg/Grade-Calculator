The Grade Calculator is a Python-based project that helps analyze and visualize student grades using Pandas for data processing and Matplotlib for graphical representation.
It allows users to:

Import or enter student marks

Calculate total and average scores

Assign grades automatically

Visualize performance trends with charts



---

⚙️ Features

✅ Import student data from a CSV file
✅ Calculate Total, Average, and Grade for each student
✅ Display summarized statistics (highest, lowest, mean, etc.)
✅ Generate bar charts, pie charts, and line graphs using Matplotlib
✅ Export the processed data to a new CSV file
